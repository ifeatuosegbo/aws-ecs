About This Page

This sample homepage is served by an Apache HTTP Server running inside a Docker container built on CentOS. You can customize this file to display your own content or application.

Try It Yourself

To replace this page, simply edit /var/www/html/index.html inside the container or map your local HTML folder using: 

docker run -d -p 8080:80 -v $(pwd)/html:/var/www/html centos-httpd
